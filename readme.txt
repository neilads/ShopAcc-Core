Plugin quản lý shop account với tính năng upload hàng loạt và tùy chỉnh WooCommerce.

== Description ==

ShopAcc Core là plugin WordPress chuyên dụng cho việc quản lý shop bán account với các tính năng:

* Upload sản phẩm hàng loạt
* Tối ưu hóa hình ảnh
* Tùy chỉnh hiển thị WooCommerce
* Quản lý sản phẩm theo nhóm
* API REST để quản lý từ xa

== Installation ==

1. Upload plugin vào thư mục `/wp-content/plugins/`
2. Kích hoạt plugin qua menu 'Plugins' trong WordPress
3. Plugin sẽ tự động tạo trang "Đăng Acc"

== Frequently Asked Questions ==

= Có cần WooCommerce không? =

Có, plugin này yêu cầu WooCommerce để hoạt động.

= Có thể upload bao nhiêu sản phẩm cùng lúc? =

Không có giới hạn, nhưng khuyến nghị không quá 100 sản phẩm mỗi lần upload.
